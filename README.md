# Churn Analysis Dashboard 

An interactive Churn Analysis Dashboard built using Microsoft Power BI, Advanced DAX, and Power Query to analyze user behavior and identify key churn drivers across demographics, subscription plans, and streaming patterns.
This dashboard helps stakeholders understand churn trends, identify high-risk users, and improve retention strategies.

# Project Objective

The goal of this dashboard is to:

Analyze user churn behavior
Identify key churn drivers
Compare active vs churned users
Analyze churn across demographics
Study impact of buffering & device type
Evaluate subscription plan churn

# Skills Used
Microsoft Power BI
Advanced DAX
Power Query
Data Analysis

Approximately one-third of users have churned, indicating retention opportunities.

# Key Insights
Overall churn rate is 31%
Younger users churn more
Higher buffering increases churn
Free plan users most likely to churn
Device type has minimal impact
Genre preference influences churn
Auto-renewal reduces churn

# Business Value

This dashboard helps:

Reduce churn rate
Improve streaming performance
Optimize subscription plans
Improve content strategy
Increase user retention
